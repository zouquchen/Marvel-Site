# 最短路径算法

## 1. 概述

对于加权图来说，最短路径是指两顶点之间经过的边上权值之和最小的路径，并且我们称路径上的第一个顶点是源点，最后一个点是终点。

## 2. 算法

### 2.1 迪杰斯特拉算法

迪科斯彻算法使用了广度优先搜索解决赋权有向图或者无向图的单源最短路径问题，算法最终得到一个最短路径树。该算法常用于路由算法或者作为其他图算法的一个子模块。
